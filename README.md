## 🚀 Start
To start, clone this repo and run this code:
```sh
python -m bot
```

## ⚙️ Configuring
Fill in .secrets.toml token for [Telegram](t.me/BotFather) and other services. For example:

```toml
[tokens]
token = "12345678909:AVEFWGRGHTHHRGGERGEG"
access_token = "rf8gmd46..."
youtube_key = "AB5C..."
todoist = "85b34485..."
```
